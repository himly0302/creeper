- **配置项**：相关的环境变量、配置文件、常量定义

### 3. 技术栈
- 编程语言及版本要求
- 核心依赖库（名称 + 版本 + 用途）
- 外部服务（数据库、消息队列、API）

### 4. 关键实现细节
- 性能优化点（如缓存策略、并发控制、批处理）
